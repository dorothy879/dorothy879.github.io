Projects

Shona to English Translator (Final Year Project) For my final year project at Bindura University of Science Education, I developed a **Shona to English Translator** using **Statistical Machine Translation (SMT)**. This project aims to assist in translating Shona, one of Zimbabwe’s native languages, into English, thus making communication more accessible between speakers of the two languages. The translator is built using SMT techniques, which involve analyzing large datasets of parallel text in both Shona and English to predict the most likely translations. This approach enables the system to handle various linguistic structures and nuances, making the translation process more dynamic and context-aware. Key aspects of the project include: - **Statistical Machine Translation (SMT)**: Leveraging probability-based models to improve translation accuracy. - **Corpus creation and management**: Collecting and preparing parallel text datasets for training the SMT system. - **Natural language processing (NLP)**: Enhancing the translator’s ability to handle different sentence structures and phrases. - **User interface design**: Developing a simple, intuitive interface for users to input Shona phrases and receive English translations. This project not only deepened my understanding of machine learning and statistical methods but also contributed to the preservation and digitization of the Shona language.
